# BERT Intent Recognition System 

This project uses **DistilBERT (Transformer-based NLP model)** to classify user intent from text queries.

## 🔹 Objective

To build an intelligent system that understands user input and predicts the correct intent (similar to Siri / Google Assistant).

## 🔹 Features

* Intent classification using BERT
* High accuracy (~95%)
* Confusion matrix & evaluation metrics
* Interactive prediction system
* Confidence visualization (bar charts)

## 🔹 Example

Input: "play music"
Output: PlayMusic 🎵

## 🔹 Tech Stack

* Python
* TensorFlow
* HuggingFace Transformers
* Scikit-learn
* Matplotlib

## 🔹 Future Scope

* Trigger real-world applications (Spotify, YouTube)
* Build chatbot / voice assistant
* Deploy using Flask / FastAPI
